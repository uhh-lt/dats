# Features & Views

Welcome to the core reference manual for the Discourse Analysis Tool Suite.

While DATS is designed to provide a seamless, unified workflow, its powerful functionalities are organized into distinct, specialized **Views**—each tailored to a specific phase of the qualitative research process.

Whether you are exploring your corpus in the Search View, deeply coding text in the Annotation View, or mapping out theoretical frameworks on the Whiteboard, this section provides comprehensive, step-by-step guidance for every single interface and button within the platform.

Use the navigation menu to explore detailed documentation on:

* **Core Workspaces:** Search, Annotation, and Logbook views.
* **Advanced AI Integration:** LLM Assistant, Custom Classifiers, and Perspectives.
* **Analysis:** Quantitative tools like Code Frequency, Timeline Analysis, and COTA.
* **Data Management:** Document Upload, Export/Import capabilities, and specialized utilities.
